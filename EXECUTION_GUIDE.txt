================================================================================
TROUBLESHOOTING
================================================================================

Problem: "javac is not recognized"
Solution: Install Java JDK and add it to system PATH

Problem: "Cannot find class main.MainApp"
Solution: Ensure you're in the correct directory when running

Problem: "MySQL Connection Error"
Solution: This is OK if MySQL is not running. The code tests still work.

Problem: Still getting NullPointerException
Solution: Delete all .class files and recompile with the RUN.bat script
